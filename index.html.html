Mis Proyectos

Proyecto 1: Portafolio Web Semántico

Desarrollo de una página web personal utilizando HTML5 semántico y sensillo, aplicando buenas prácticas de estructura, jerarquía de encabezados y validación según estándares web.

Este proyecto permitió reforzar conceptos como el uso correcto de etiquetas semánticas, organización del contenido y accesibilidad.

Proyecto 2: Gestión de Base de Datos

Proyecto académico enfocado en el diseño y consulta de bases de datos, utilizando sentencias SQL, vistas, joins y control de permisos.

Se trabajó sobre escenarios reales, aplicando consultas complejas y buenas prácticas de modelado de datos.

Proyecto 3: Proyecto Tecnológico Integrador

Desarrollo de un proyecto integrador donde se combinaron conocimientos de análisis de sistemas, lógica de programación y documentación técnica.

El enfoque principal fue la solución de una necesidad específica mediante el uso de herramientas tecnológicas.
